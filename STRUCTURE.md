# Monorepo Structure

- `/apps/customer_app`: Flutter App for Customers.
- `/apps/restaurant_app`: Flutter App for Restaurants.
- `/apps/rider_app`: Flutter App for Delivery Riders.
- `/apps/admin_web`: React (Vite) Dashboard for Admins.
- `/firebase`: Configuration, Rules, and Cloud Functions.
- `/packages/shared_core`: Shared models and utilities (Dart).
